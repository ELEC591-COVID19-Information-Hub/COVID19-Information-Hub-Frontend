# COMP 590 / ELEC 591 Covid-19 Information HUB

## Build
To run this project:
```shell
npm install
npm start
```

To build & deploy this project:
```shell
npm run build
cd build
surge --domain <your-surge-domain>
```

## Deployment
This project is deployed to surge at https://covid-19-info-hub.surge.sh.
